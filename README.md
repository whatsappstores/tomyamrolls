# TomYam Rolls — сайт с заказом через WhatsApp

Статичный сайт (HTML/CSS/JS, без сборки и сервера) под бренд **TomYam Rolls**:
цвета и логотип взяты из вашего лого, меню разбито по категориям, а оформление
заказа идёт прямо в WhatsApp.

## Что внутри
- `index.html` — разметка страницы
- `style.css` — стили (палитра: чёрный `#101010`, красный `#C1272D`, белый)
- `script.js` — меню, корзина, сборка ссылки на WhatsApp
- `assets/logo.png` — ваш логотип (уже подключён в шапке, подвале и favicon)
- `images/` — папка для фотографий (см. ниже)
- Корзина сохраняется в localStorage браузера — не теряется при обновлении страницы

## 1. Добавьте фотографии

Сайту специально оставлено много места под фото — блюда, зал, кухню. Пока
фотографий нет, вместо них показывается аккуратная подпись с названием файла,
которое нужно положить в папку `images/`. Как только файл появится — подпись
сама пропадёт и покажется фото.

| Где на сайте | Файл | Рекомендуемый размер |
|---|---|---|
| Главный экран (круглое фото) | `images/hero.jpg` | квадрат, от 900×900 |
| Блок «О нас» | `images/about.jpg` | вертикальное, 800×1000 |
| Карточки блюд | `images/{id}.jpg` — см. таблицу ниже | 4:3, от 800×600 |

ID блюд для фото (соответствуют `MENU` в `script.js`):
```
Закуски:            ap1
Роллы:               rl1 rl2 rl3 rl4 rl5
Темпура-роллы:       tp1 tp2 tp3
Запечённые роллы:    bk1 bk2
Маки:                mk1 mk2 mk3 mk4
Суши и гункан:       su1 su2 su3 su4
Поке:                pk1 pk2
Супы:                sp1
Салаты:              sl1
Сеты:                se1 se2 se3 se4 se5
```
Например, фото для «Суп Том Ям» — это `images/sp1.jpg`.

## 4. Как запустить через GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `tomyam-rolls-site`).
2. Загрузите в него все файлы и папки (`index.html`, `style.css`, `script.js`,
   `assets/`, `images/`) через веб-интерфейс «Add file → Upload files» или git:
   ```bash
   git init
   git add .
   git commit -m "Первая версия сайта TomYam Rolls"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/tomyam-rolls-site.git
   git push -u origin main
   ```
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`, папка `/ (root)`
5. Нажмите **Save**. Через 1–2 минуты сайт будет доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/tomyam-rolls-site/`

Дальше просто редактируйте файлы и делайте `git push` — GitHub Pages подхватит
изменения автоматически.

## Как устроена отправка заказа в WhatsApp

При нажатии «Оформить заказ» сайт формирует текстовое сообщение со списком
выбранных блюд, количеством и итоговой суммой, адресом и комментарием — и открывает
`https://wa.me/<номер>?text=<сообщение>` в новой вкладке. Пользователю остаётся
только нажать «Отправить» в самом WhatsApp — заказы приходят прямо в чат, без
сервера и без базы данных.
